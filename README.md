# n-buna 全楽曲データベース

n-buna（ナブナ）の全楽曲をまとめたファンメイドの非公式データベース。
ボカロ曲・ヨルシカ・楽曲提供・コラボ・アレンジ等を網羅し、Webページとして公開している。

- **公開URL:** https://sonicpotato08.github.io/nbuna-db/
- **作者:** たんそ ([@carbon_nanoja](https://x.com/carbon_nanoja))

---

## アーキテクチャ

```
GitHub Pages (静的ホスティング)
  └── index.html (単一ファイル、HTML/CSS/JS)
        └── fetch() → Google Sheets CSV公開URL
```

- index.htmlはリポジトリのルートに配置
- データはGoogleスプレッドシートから毎回fetchで取得（ページリロード時に最新化）
- ビルド不要の純粋な静的HTML。GitHub Pagesの「Deploy from a branch」でmainブランチからデプロイ
- 外部ライブラリ依存なし（Vanilla JS）

### データフロー

```
Googleスプレッドシート（手動編集）
  ↓ ウェブに公開（CSV形式）
  ↓ index.html内のJSがfetchでCSVを取得
  ↓ クライアントサイドでパース・レンダリング
```

スプレッドシートを編集 → 5分以内にCSV公開に反映 → ページリロードで表示更新。
データのみの更新はスプレッドシートを編集するだけでよく、git pushは不要。

## スプレッドシート仕様

### 列定義（この順序・列名を変更しないこと）

| 列名 | 内容 | 例 |
|---|---|---|
| 曲名 | 楽曲タイトル | ウミユリ海底譚 |
| ボーカル | 歌唱者。インスト曲は「—」 | 初音ミク |
| 収録アルバム | 収録先。複数ある場合は「 / 」区切り | ニコニコ動画投稿 / カーテンコールが止む前に |
| 配信日 | YYYY-MM-DD形式。不明な場合は空欄 | 2014-02-10 |
| カテゴリ | 下記の10種類から選択 | オリジナル |
| リンク | ニコニコ/Spotify/YouTube等のURL | https://www.nicovideo.jp/watch/sm22960446 |
| 配信状況 | 下記の4種類から選択 | ◯ |
| データソース | 情報の出典 | nicodb,wikipedia_nbuna |
| 備考 | 確定した事実情報のみ記載 | 同人CD(DUED-1004) |

### カテゴリ一覧

| カテゴリ | 意味 |
|---|---|
| オリジナル | n-bunaが作詞作曲した楽曲 |
| 楽曲提供 | 他アーティストへの書き下ろし提供 |
| コラボ・参加 | ギター参加、共作、feat.等 |
| アレンジ | 自作曲のセルフアレンジ |
| リミックス | 他者曲のリミックス |
| コンピ収録 | 既存曲のコンピレーション再収録 |
| カバー・引用 | 他者がn-buna曲をカバーしたもの |
| ゲーム | ゲーム関連楽曲 |
| 別バージョン | 既存曲の別ボーカル版等 |
| ヨルシカ | ヨルシカ名義の全楽曲 |

### 配信状況

| 値 | 意味 |
|---|---|
| ◯ | サブスク/YouTube等で配信確認済み |
| ✕ | 配信がないことを確認済み（購入限定） |
| — | 未確認 |
| 対象外 | 他者によるカバー等、n-bunaの配信判定対象外 |

### 備考の記載ルール

- 確定した事実のみ記載する（推測・可能性・要確認等の表現は使わない）
- クレジット内容（作詞:YUKI, 作曲・編曲:n-buna等）
- 収録トラック番号（Track 03等）
- タイアップ情報
- インスト曲は「インスト」と記載

## 機能

- **テキスト検索:** 曲名・ボーカル・アルバム名・カテゴリ・備考を横断検索
- **カテゴリフィルター:** チップクリックで絞り込み（複数選択可、トグル）
- **配信状況フィルター:** 同上
- **全列ソート:** ヘッダークリックで昇順/降順トグル
- **デフォルトソート:** 配信日の昇順（古い順）。配信日が空の行は常に末尾
- **統計表示:** Total Tracks / Streaming / Limited のカウント
- **最新データ表示:** CSV内の最新配信日をフッターに表示
- **リンク:** クリックでニコニコ/Spotify/YouTube等に遷移

## JS内の主要変数・定数

| 変数名 | 説明 |
|---|---|
| `SHEET_CSV_URL` | GoogleスプレッドシートのCSV公開URL |
| `COL` | CSVの列名マッピング |
| `CAT_LABELS` | カテゴリ値 → 表示ラベル |
| `CAT_CSS` | カテゴリ値 → CSSクラス |
| `DATA` | fetchしたCSVをパースしたオブジェクト配列 |

## デプロイ

### 更新手順

**ページの更新:** index.htmlを変更した場合のみgit pushが必要

---

## 注意事項

- スプレッドシートの列名（1行目ヘッダー）を変更するとページが壊れる
- カテゴリ名を新規追加する場合はindex.htmlの`CAT_LABELS`/`CAT_CSS`にも追加が必要
- 配信状況の値は「◯」「✕」「—」「対象外」の4種のみ使用すること
